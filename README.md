# DramaCraft
本作品提出"叙事规则内生驱动的剧本智能创作新范式"，针对AI剧本生成中长文本叙事一致性失控、结构化控制不足、人机协同深度缺乏等核心痛点，构建了"题材结构配置+规则校验引擎+大模型生成执行"三位一体联动架构。  系统将创作流程分解为核心设定、人物设定、剧情大纲、分幕正文四个阶段，嵌入硬性校验与软性提示的双层规则校验引擎，并通过双轨协同编辑界面与当前幕AI修订闭环机制，实现从一次性生成工具到结构化可控创作伙伴的转变。  系统基于Vue3与FastAPI的B/S架构实现，采用Neo4j图数据库支撑叙事结构可视化，支持电影、电视连续剧、短剧三类题材的差异化创作，面向专业编剧、影视公司、短视频创作者及文学爱好者等多元用户，在提升创作效率与保障叙事质量方面具有显著应用价值。
